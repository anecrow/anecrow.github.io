游玩

创作

制作并上传内容之前, 还需要做一些准备:

  • Unity
    • 指定版本: 2019.4.31f1
      • Unity: 使用Hub下载
  • VRC SDK3

    SDK2 已不再推荐使用

    • Avartar 角色
      • Download SDK3 - Avatars
    • World 场景
      • Download SDK3 - Worlds
  • VRC账户

    使用独立邮箱注册账户(Steam账户不可以)
    可选绑定Steam账户

    • Trust Rank: New User

      意味着纯新账户是不能使用创作功能的
      包括 上传和测试

      提升等级的具体方式并未明说
      但多与人互动添加好友逛不同地图等方法有一定积极作用(单纯挂机用处有限)

配置开发环境

显然 SDK 分为了 Avatars 和 Worlds
意味着 VRC 针对两者的开发环境做出了区分

将两个SDK载入同一个项目中会导致错误

SDK 以 .unitypackage 文件形式打包
在项目中打开该文件将会弹出对应导入窗口

更多配置开发环境细节参阅 官方文档: Setting up the SDK
