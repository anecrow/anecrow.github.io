静态站点生成器

这是啥

HUGO 是个开源易用快速可扩展的轻型静态站点生成器

有啥用

简单来说就是可以把你的 Markdown 文章库渲染成漂亮的静态 HTML 站点(当然还需要部署成 Web 服务器, 比如 Github Page)

需要什么准备

取决于你要做什么:

  1. 单纯的文本写作
    • 简单的 Markdown语法
      • 关注内容的沉浸式写作, 让复杂的 Word 见鬼去吧
    • 选择一个漂亮的 Theme主题
      • 一键将写好的文档编译成完整的小站
    • 合适的文本编辑器
      • 记事本 (你愿意的话)
      • 专用 Markdown编辑器 (随便搜一个)
      • 轻度开发者可以直接用 VS Code
    • 简单安装使用入门
  2. 定制化个人小站
    • 基本的 前端 知识: HTML, CSS, JavaScript
      • 至少需要理解 Web页面 是如何被控制和呈现的
    • Hugo框架 知识: Hugo, Go Template
      • 明白 Hugo框架 是如何分类和组织内容的
    • 不为零的 美术 和 设计 知识: 自备
      • 在能用的基础上追求美感和用户体验
    • 从新建空白骨架开始

    从安装到日常使用的非开发向内容.
