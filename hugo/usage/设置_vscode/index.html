插件

推荐

  • Markdown All in One

  • Markdownlint

可选

配置文件支持和语法高亮

  • TOML: Better TOML
    • 推荐使用, 语法简洁标准化, 歧义少
  • YAML: YAML
    • 语法简单, 应用广泛, 但用户不规范编写会产生不可预知的问题, 排查稍微麻烦
  • JSON(不推荐)
    • 手写有点啰嗦, VSC 自带
  • XML(不推荐)
    • 不适合一般人类编写与理解
  • 全都要(反正其他项目多少也会用到)

工作区

为所有站点项目提供总览, 方便管理

设置主根目录

提供 hugo 命令初始环境

使用 VSC 打开文件夹 选择 Hugo库 文件夹
参照上一章使用的是 hugo_sites/, 原地保存为工作区

核心 hugo.exe 文件处在工作区根目录 生成的小站处于下一级目录中

添加小站目录

将小站纳入环境

右键 VSC 资源管理器工作区空白处, 选择 将文件夹添加到工作区
添加之前生成的 Hugo小站 hugo_sites/new_site/ 文件夹

其他路径的 Hugo小站 也可以添加进此工作区, 方便统一管理

Tasks

使用 VSC 的自定义 Tasks 功能, 包装 Hugo命令 快捷一键使用

配置

太长不看版

使用已写好的配置文件: workspace.code-workspace
覆盖 Hugo库 工作区根目录下的 workspace.code-workspace 文件内容即可

手动操作版

Tasks 编写语法与逻辑1 2: VSCode Task 自定义任务

按下 F1 打开 VSC命令栏, 输入 task, 选择 任务: 打开工作区任务
选择 Others 运行任意外部命令的示例
将会自动打开 workspace.code-workspace 文件

此时会自动在工作区配置文件内生成 tasks 示例配置:

{
  "folders": [
    {
      "path": "."
    }
  ],
  "settings": {},
  "tasks": {
    "version": "2.0.0",
    "tasks": [
      {
        "label": "echo",
        "type": "shell",
        "command": "echo Hello"
      }
    ]
  }
}

将 Hugo核心路径 设置为变量 ${workspaceFolder}\\hugo, 添加进 settings:

{
  "settings": {
    "hugo": "${workspaceFolder}\\hugo",
  }
}

将常用 Hugo命令 编写为 Task
修改 tasks 如下:

{
  "tasks": {
    "version": "2.0.0",
    "problemMatcher": [],
    "options": {
      "cwd": "${fileWorkspaceFolder}"
    },
    "tasks": [
      {
        "label": "新建站点项目",
        "type": "shell",
        "command": "${config:hugo} new site ${input:newSiteName}",
        "options": {
          "cwd": "${workspaceFolder}"
        },
      },
      {
        "label": "新建站点项目并加入工作区",
        "type": "shell",
        "command": "echo TODO", // TODO: 新建站点项目并加入工作区
        "options": {
          "cwd": "${workspaceFolder}"
        },
      },
      {
        "label": "运行当前站点",
        "type": "shell",
        "command": "${config:hugo} server ${input:serverFlags}",
      },
      {
        "label": "构建当前站点",
        "type": "shell",
        "command": "${config:hugo} ${input:serverFlags}",
      },
      {
        "label": "在当前站点新建主题",
        "type": "shell",
        "command": "${config:hugo} new theme ${input:newThemeName}",
      },
      {
        "label": "在当前站点新建页面",
        "type": "shell",
        "command": "${config:hugo} new ${input:sectionName}\\${input:newPageName}.md",
      },
      {
        "label": "在当前位置新建页面",
        "type": "shell",
        "command": "echo TODO", // TODO: 在当前位置新建页面
      },
      {
        "label": "在当前位置新建Section",
        "type": "shell",
        "command": "echo TODO", // TODO: 在当前位置新建Section
      },
      {
        "label": "在当前位置新建Leaf",
        "type": "shell",
        "command": "echo TODO", // TODO: 在当前位置新建Leaf
      },
    ],
    "inputs": [
      {
        "type": "promptString",
        "id": "newSiteName",
        "description": "输入新建站点名称",
        "default": "new_site"
      },
      {
        "type": "promptString",
        "id": "serverFlags",
        "description": "命令参数",
        "default": "-D"
      },
      {
        "type": "promptString",
        "id": "buildFlags",
        "description": "命令参数"
      },
      {
        "type": "promptString",
        "id": "sectionName",
        "description": "section路径",
        "default": "post"
      },
      {
        "type": "promptString",
        "id": "newPageName",
        "description": "页面标题",
        "default": "new_post"
      },
      {
        "type": "promptString",
        "id": "newThemeName",
        "description": "输入新建主题名称",
        "default": "new_theme"
      },
    ]
  }
}

执行

F1 选择 任务:运行任务
点击 对应任务 执行

新增的任务如下:

  • 新建站点项目 > 输入名称
    • 在Hugo库根目录新建站点
    • hugo new site <name>
    • 输入项:
      • <name>
  • 新建站点项目并加入工作区 > 输入名称
    • (未实现)
    • 在Hugo库根目录新建站点并自动将项目文件夹加入工作区
    • hugo new site <name>
    • 输入项:
      • <name>
  • 运行当前站点
    • hugo server
  • 构建当前站点
    • hugo
  • 在当前站点新建主题 > 输入名称
    • 在当前站点创建自定义主题theme
    • hugo new theme <name>
    • 输入项:
      • <name>
  • 在当前站点新建页面 > 输入 Section 路径 > 输入名称
    • 在当前站点创建新博文内容页
    • hugo new <section_name>/<page_name>.md
    • 输入项:
      • <section_name>
      • <page_name>
  • 在当前位置新建页面 > 输入名称
    • (未实现)
    • 在当前站点的当前目录创建新博文内容页
    • hugo new <section_current>/<page_name>.md
    • 输入项:
      • <page_name>

可以根据需求自行修改
