侧边菜单栏

优化体验和反馈.

增大点击区域

因为 <a> 元素本身是行内元素, 列表项只有标题文字本身是可点击的, 范围比较小
只需要在内部放入块级元素, 就能将它撑开而获得更大的点击区域

<a href="{{.Site.BaseURL}}">
    <h1>Home</h1>
</a>

或者使用内联元素, 在样式表中指定 display: block;

<a href="{{.Permalink}}">
    <span class="nav-item">
        {{.Title}}
    </span>
</a>

还可以利用已有样式做二级菜单项

<a href="{{.Permalink}}">
    <code class="nav-item">
        {{.Title}}
    </code>
</a>

构造菜单项

为了方便导航, 将当前层级看作二级菜单项, 父级层级看作一级菜单项进行构造

taxonomy 分类页暂时使用 .Site.Taxonomies

{{ $section := .CurrentSection }}
{{ if .CurrentSection.Parent }}
    {{ $section = .CurrentSection.Parent }}
{{ else if eq .Kind "taxonomy" }}
    {{ $section = .Site.Taxonomies }}
{{ end }}
<nav id="side-nav">
    <a href="{{ .Site.BaseURL }}">
        <h1>Home</h1>
    </a>
    {{ range $section.Pages.Reverse }}
        <a href="{{ .Permalink }}">
            <span class="nav-item">
                {{ .Title }}
            </span>
        </a>
        {{ range .Pages.Reverse }}
            <a href="{{ .Permalink }}">
                <code class="nav-item nav-subitem">
                    {{ .Title }}
                </code>
            </a>
        {{ end }}
    {{ end }}
</nav>

确定一级菜单对象后, 进行两层迭代

注意在语法中创建并赋值变量使用 := 符号, 而重新赋值时使用 = 符号

当前菜单项

额外地, 可以让当前页面所在的导航项变得不可用, 也是一种指示信息
使用 {{ if }} 语句选择性地给当前项包裹 <a> 元素就可以实现这个功能

{{ $section := .CurrentSection.Parent }}
{{ if eq .Kind "home" }}
    {{ $section = .CurrentSection }}
{{ end }}
{{ $url := .Permalink }}
<nav id="side-nav">
    <a href="{{ .Site.BaseURL }}">
        <h1>Home</h1>
    </a>
    {{ range $section.Pages.Reverse }}
        {{ if eq .Permalink $url }}
            <span class="nav-item nav-current">
                {{ .Title }}
            </span>
        {{ else }}
            <a href="{{ .Permalink }}">
                <span class="nav-item">
                    {{ .Title }}
                </span>
            </a>
        {{ end }}
        {{ range .Pages.Reverse }}
            {{ if eq .Permalink $url }}
                <code class="nav-item nav-subitem nav-current">
                    {{ .Title }}
                </code>
            {{ else }}
                <a href="{{ .Permalink }}">
                    <code class="nav-item nav-subitem">
                        {{ .Title }}
                    </code>
                </a>
            {{ end }}
        {{ end }}
    {{ end }}
</nav>

给当前页面项指定 nav-current 类, 以便后续指定样式

格式美化

首先定义一般情况的样式

.side-container h1,
.nav-item {
    padding-left: var(--gap-unit) !important;
}

.nav-item {
    display: block;
    margin: 2px 0 !important;
    padding-top: 0.6em !important;
    padding-bottom: 0.4em !important;
}

--gap-unit: 16px

接着对鼠标行为 hover 进行微调

.side-container a:hover {
    text-decoration: none !important;
}

.side-container h1:hover,
.nav-item:not(.nav-current):hover {
    font-weight: bold;
    color: var(--color-accent-emphasis);
}
  • 属性 text-decoration: none 不显示下划线
  • 选择器 .nav-item:not(.nav-current) 排除当前页面项

TOC

将 TOC 贴在屏幕下边缘, 位置随窗口滚动

#TableOfContents {
    position: fixed;
    bottom: 0;
}

#TableOfContents>ul {
    padding-left: var(--gap-unit);
}

同样的也可以设置 侧边菜单栏 为 sticky, 窗口滚动超出位置时随窗口滚动

#side-nav {
    position: sticky;
    top: 0;
}

注意: sticky 物体当尺寸超出窗口的时候, 需要一直划到底部才能看到超出的部分
