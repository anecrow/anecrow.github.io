页眉和页脚可以提供额外信息.

Header

页眉一般用来放置当前文章相关的信息

在 partials/header.html 中写入

<header>
    <h1>{{ .Title }}</h1>
    {{ if eq .Kind "page" }}
        <p>{{ .Date.Format "2006/01/02 15:04:05" }}</p>
        <p>{{ .Lastmod.Format "2006/01/02 15:04:05" }}</p>
        <p>{{ .WordCount }}</p>
        <p>{{ .ReadingTime }}min</p>
        <hr/>
    {{ end }}
</header>

就能让所有页面的 header 部分拥有一个标题

而类型为 page 也就是文章页面会额外展示创建日期, 修改日期, 字数和预估阅读时间

面包屑导航1

根据官方示例, 修改一个横向的版本

<nav class="breadcrumbnav">
    {{ template "breadcrumbnav" (dict "p1" . "p2" .) }}
</nav>

{{ define "breadcrumbnav" }}
    {{ if .p1.Parent }}
        {{ template "breadcrumbnav" (dict "p1" .p1.Parent "p2" .p2 ) }}
    {{ else if not .p1.IsHome }}
        {{ template "breadcrumbnav" (dict "p1" .p1.Site.Home "p2" .p2 ) }}
    {{ end }}

    {{ if eq .p1 .p2 }}
        <code>{{ .p1.Title }}</code>
    {{ else }}
        <span>
            <a href="{{ .p1.Permalink }}">
                <code>{{ if eq .p1.Kind "home" }}Home{{ else }}{{ .p1.Title }}{{ end }}</code>
            </a>
        </span>
        <span>></span>
    {{ end }}
{{ end }}

使用 {{ template }} 语法实现的迭代循环

在 header.html 中插入该 partial

<header>
    <h1>{{ .Title }}</h1>
    {{ partial "breadcrumb.html" . }}
</header>

页面信息

建立容器元素 blockquote

{{ $dateFormat := "2006/01/02 15:04:05" }}
<header>
    <h1>{{ .Title }}</h1>
    {{ partial "breadcrumb.html" . }}
    <blockquote id="page-info">
        <span><code>Date</code>: {{ .Date.Format $dateFormat }}</span>
        <span><code>Lastmod</code>: {{ .Lastmod.Format $dateFormat }}</span>
        <br>
        {{ if eq .Kind "page" }}
            <span><code>WordCount</code>: {{ .WordCount }}</span>
            <span><code>ReadingTime</code>: {{ .ReadingTime }}min</span>
        {{ else }}
            <span><code>Pages</code>: {{ len .Pages }}</span>
            <span><code>Sections</code>: {{ len .Sections }}</span>
        {{ end }}
    </blockquote>
</header>

设置样式

#page-info {
    background-color: var(--color-neutral-muted);
    margin: 1em 0;
    padding: 1em;
    border-radius: 6px
}

Footer

页脚放置小站的额外信息

前后页面导航

在 partials/footer.html 中写入

<footer>
    <div class="flex-container">
        <div>
            {{ template "otherPage" (dict "page" .PrevInSection "label" "<< Prev") }}
        </div>
        <div></div>
        <div>
            {{ template "otherPage" (dict "page" .NextInSection "label" "Next >>") }}
        </div>
    </div>
    <hr>
</footer>

{{ define "otherPage" }}
    {{ if .page }}
        <a href="{{ .page.Permalink }}">
            <div>
                <h2>{{ .label }}</h2>
                <span>{{ .page.Title }}</span>
            </div>
        </a>
        <p>{{ .page.Summary}}</p>
    {{ end }}
{{ end }}
