角色配饰操作组件

Lyra框架 对于角色展示的核心组件
负责角色 模型|装备|配饰 的添加和删除

类型接口

LyraCharacterPart

角色部件对象结构

  • 属性
    • PartClass
      • 任何Actor类型
      • 通常使用拓展的 LyraTaggedActor 类
    • SocketName
      • 目标插槽名称
      • 放置角色时留空: None
        • 设置角色
      • 放置配饰时写对应槽位
        • 设置配饰
    • CollisionMode
      • 碰撞模式

主要功能函数

从 Lyra 示例项目的游戏性配置使用的 Pawn 可以发现
这些 LyraPawnData -> Pawn Class 配置中的骨骼网格体位置都是留空的

需要调用 AddCharacterPart 函数添加模型

LyraCharacterPart 类型的 PartClass 属性可以是任意 Actor 类型
为了配合 Lyra框架 可以使用框架拓展出的 LyraTaggedActor 类

LyraTaggedActor 类仅是给 Actor 类增添了标签属性, 其他没有区别
