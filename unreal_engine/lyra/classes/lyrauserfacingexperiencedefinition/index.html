关卡配置定义类.

非常离谱长还难以理解的命名
或许可以将它简化理解为 MapDefinition

纳入资产扫描
站内文章: 关卡配置

配置参数

Unreal Projects/xxx/Source/LyraGame/GameModes/LyraUserFacingExperienceDefinition.h

  • Experience
    • Map ID 指定地图
      • 资产槽类型: Map 关卡
        • 只能放入指定扫描的资产
        • 站内文章: 关卡资源
      • 将会依照这个目标加载关卡
    • Experience ID 指定游戏体验
      • 资产槽类型: LyraExperienceDefinition
        • 只能放入指定扫描的资产
        • 站内文章: 游戏性配置
      • 将会依照这个目标对关卡设置 Experience
      • 站内文章: LyraExperienceDefinition
    • Extra Args
      • 略过
    • Tile Title 卡片标题
      • 字符串
      • 作为关卡列表卡片标题
    • Tile Sub Title 卡片副标题
      • 字符串
      • 作为关卡列表卡片副标题
    • Tile Description 卡片说明
      • 字符串
      • 作为关卡列表卡片说明
    • Tile Icon 卡片封面
      • 资产槽类型: Texture 纹理
      • 作为关卡列表卡片封面
    • Is Default Experience 快速游戏
      • 布尔
      • 为 True 时作为快速游戏的地图之一, 并在列表中优先展示
    • Show in Front End 主菜单展示
      • 布尔
      • 是否在主菜单展示
    • Record Replay
      • 略过
    • Max Player Count 最大玩家
      • 整数
      • 最大玩家数
  • Loading Screen
    • Loading Screen Widget 加载画面
    • 资产槽类型: Widget UI控件
    • 作为进入或退出关卡时的加载画面使用
