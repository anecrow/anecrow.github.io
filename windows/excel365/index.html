365订阅版

新增了可以称为革命性更新的特性
将原本的操作流有了完全不一样的体验

动态数组

简单说就是可以将单元格内的一组值自动溢出到相邻格


动态数组公式和溢出的数组行为

以公式所在单元格为数组左上角原点进行溢出
将数据副本填充到对应相邻单元格中

同时增添了运算符#, 引用原点格时使用关键字可以获取原数组


溢出范围运算符

相比旧版使用特殊大括号的数组语法更加直观且方便

更多介绍与实例:数组公式指南和示例
可以将数组视为值的行或列，或值行和列的组合。
数组公式可以返回多个结果或单个结果。
从 Microsoft 365 的 2018 年 9 月更新开始，任何可返回多个结果的公式都将自动向下或溢出到相邻单元格中。

无论 动态数组公式 是使用现有函数还是动态数组函数，都只需输入到单个单元格中，然后按 Enter 进行确认。
以前，旧数组公式需要先选择整个输出区域，然后使用 Ctrl+Shift+Enter 确认公式。
它们通常称为 CSE 公式。

LET 与 LAMBDA

更进一步地, 新版中支持革命性的 变量 与 λ函数
极大颠覆了旧版中需要使用额外单元格保存常用的计算中间值和不停复制粘贴那些复杂到无法阅读公式的状况

LET 函数会向计算结果分配名称。 这样就可存储中间计算、值或定义公式中的名称。 这些名称仅可在 LET 函数范围内使用。

不用再记住特定范围/单元格引用是指什么、你的计算在执行什么操作，也不用再复制/粘贴相同的表达式。 借助可声明和命名变量的能力，你可为自己和公式使用者提供有意义的上下文。
LET 函数

使用 LAMBDA 函数创建可重用的自定义函数，并使用易记名称调用它们。

可以为常用公式创建函数，无需复制并粘贴此公式(可能容易出错)，并有效地将自己的函数添加到本机 Excel 函数库中。
此外，LAMBDA 函数无需 VBA、宏或 JavaScript，因此非程序员也可以从使用中获益。
